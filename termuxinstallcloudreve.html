ZeroTermux安装Cloudreve


Zerotermux install Cloudreve


第一部 下载ZeroTermux

打开这个链接，找到 Assets 下面的 ZeroTermux-版本号.apk，版本号会随着版本更新变化，请及时更新。

第二部 切换源

下载完打开并安装，安装完后打开,按一下音量加（或者双击最左侧）点击切换源→清华源，等它跑完，中途遇到询问按Y回车。

第三部 安装必要组件

输入 pkg install wget git zip vim curl -y 并回车，等待跑完即可

第四部 安装Cloudreve

1.下载文件

打开这个链接，找到Assets下的cloudreve_版本号_系统_架构.tar.gz

打开你的Termux,输入 uname -a 输出了以下内容（不同手机内容不同，但大同小异）

Linux localhost 4.19.152-perf+ #1 SMP PREEMPT Thu Oct 12 21:21:14 CST 2023 aarch64 Android

Linux是系统 aarch64是架构（aarch64=arm64 armv7=arm32 其他的请自行搜索，然后按照刚才链接里的文件名选择）回到刚才链接，根据输出的选择文件，然后复制它的链接，回到ZeroTermux。

回到Termux 输入 mkdir cloudreve && cd cloudreve 再输入 wget 刚刚复制的链接

等它跑完，输入ls

会输出一个文件名复制它，在输入tar -zxvf 文件名

输出几个文件名，此时输入chmod +x cloudreve对它进行授权

这时候打开你的消息栏，应该有一个Termux的通知，展开它，点击获取唤醒锁，然后会弹出来一个弹窗，别管内容是什么，选表示允许的那个按钮就行。

打开浏览器，输入 127.0.0.1:5212

出现了cloudreve登录，按照给的账号和密码登录。登录进去之后点右上角的头像（手机版可能是先打开三个横再点头像。）然后会出现确定站点URL设置点更改，然后点击用户，在 admin 那一栏找到一个铅笔图标，点一下，找到密码，修改为自己记得住的，在点保存，以后登录直接用修改后的密码

之后就可以按 docs.cloudreve.org里的内容进行更多设置了，也可以

自己上网寻找

在Cloudreve修改设置一定要按保存!!!!

第六部 以后启动

回到termux,按下最下面小键盘的 CTRL 再按键盘上的 c （也就是Ctrl C）结束

然后我们写个快速启动脚本

输入 cd && vi scr.sh 进入编辑器，按一下 i 在里面输入 cd && cd cloudreve && ./cloudreve 输完后按小键盘上的 ESC 输入 :wq 回车，在输入chmod 777 scr.sh 以后就可以使用cd && ./scr.sh

快速启动cloudreve了，网址就是ip:5212
